# GeoStudio Console 작업 규칙

## 절대 규칙
1. **지시한 것만 수정** — 지시하지 않은 파일, 코드, 설정은 절대 건드리지 않는다
2. **수정 범위 엄수** — 리팩/수정 작업 시 해당 이슈에 명시된 부분만 변경한다
3. **추가 수정 필요 시 보고** — 작업 중 다른 파일/코드 수정이 필요하다고 판단되면, 직접 수정하지 말고 반드시 보고 후 승인받는다
4. **임의 리팩 금지** — "이왕 하는 김에", "같이 정리하면 좋겠다" 식의 임의 수정 절대 금지
5. **기존 동작 보존** — 수정 후 기존 기능이 깨지지 않는지 확인한다
